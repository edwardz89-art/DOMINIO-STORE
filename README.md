# DOMINIO STORE - Vite + React + Tailwind

Proyecto listo para desplegar en Vercel.

## Pasos rápidos
1. Instala dependencias:
   ```bash
   npm install
   ```
2. Inicia en desarrollo:
   ```bash
   npm run dev
   ```
3. Construir:
   ```bash
   npm run build
   ```
4. Subir a Vercel: conecta tu repo o sube el project ZIP. Predeterminado: dominiostore.vercel.app
